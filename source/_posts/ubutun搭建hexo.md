## 安装nodejs

`sudo apt install nodejs`

## 安装npm

`sudo apt install npm`

## 改用taobao的npm源

`npm install -g cnpm --registry=https://registry.npm.taobao.orrg` 

## 安装hexo

`cnpm install hexo-cli -g`  #-g 表示全局安装
