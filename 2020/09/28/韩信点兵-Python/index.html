range()

range()内建函数的一般格式为：

range([start,]stop[,step])

range()函数有三个参数：start和step是可选的，start表示开始，默认值为0；end表示结束；step表示每次跳跃的间距，默认值为1。函数功能是生成一个从start参数的值开始，到end参数的值结束（但不包括end）的数字序列。
